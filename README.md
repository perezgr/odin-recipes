# Odin Recipes

This project is part of the Foundations course from The Odin Project. The goal of this project is to build a basic recipe website using only HTML.

The website consists of a main index page that links to multiple recipe pages. Each recipe page includes a description of the dish, a list of ingredients, and step-by-step instructions for preparing the recipe.

Through this project, I practiced fundamental HTML concepts such as:
- Creating a basic HTML page structure
- Using headings, paragraphs, and images
- Creating links between pages
- Organizing content with ordered and unordered lists
- Structuring a small multi-page website

This project focuses on strengthening core HTML skills. Styling with CSS will be added later in the course.
